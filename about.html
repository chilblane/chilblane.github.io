recent work

Axosoft product site

www.axosoft.com

Axosoft.com is the marketing site for the Axosoft product, but includes some company-specific pages. As project manager and lead designer, I worked with other members of the marketing team to create designs, copy, graphics, and code.

Axosoft.com was the first project to use a new CSS framework I wrote called Axostyle. This framework is now in place across many of Axosoft's web properties.

The website, especially the index page and the trial signup page, underwent constant improvement. I used Google Analytics and other tools to look for new opportunities to improve the lead acquisition funnel.

Axosoft blog

blog.axosoft.com

The Axosoft blog redesign was a fun and challenging project. My goal was to modernize the home of Axosoft's new content strategy, while adhering to the Axostyle principles that established their online brand. I worked with the in-house Wordpress developer to come up with a theme that was not only a great experience for readers, but easy to use for editors, too.

Axosoft support site

support.axosoft.com

Axosoft had a pressing need for product documentation that was easy to navigate and search, and easy to maintain solely by their support team. We identified a static site generator and server that compiled Markdown articles into pages and adapted it to Axostyle.

A matching GitKraken support site was later released.

GitKraken product site

www.gitkraken.com

GitKraken.com is the marketing site for the new Git GUI app. From an aesthetic perspective, this site was very experimental and unusual to capture the GitKraken branding. It was also an ongoing design challenge as its requirements and content grew based on the product's needs.
