I’m a web designer and developer focused on marketing and product sites, based in Scottsdale, Arizona, and currently looking for new opportunities.

What I do:

Marketing web design

Front-end development

My extensive experience with HTML and CSS enables me to write effective and flexible styling frameworks and pages. I prioritize performance, progressive enhancement, and maintainability.

Cross-departmental UX

User experience isn't contained to just your website and app. I've worked hard to ensure any time a user interacts with a brand, they receive the same fantastic experience, be it transactional emails, social media, documentation, and more.

Selected work:

WoW Armory Redesign

UX Design Unsolicited Redesign Prototyping

I explored what the World of Warcraft Armory pages might look like using the new design and code of the recently launched worldofwarcraft.com site. Initially, I mocked up a design in Sketch and posted it on the /r/wow subreddit, which recently many positive reviews. I then took the existing CSS and Javascript from the WoW site and built an HTML prototype of the Armory page, approximating as much as possible what Blizzard's Web & Mobile team would actually build.

Axosoft.com

Marketing Web Design CSS Architecture Conversion UX jQuery

The website for Axosoft was my primary project while working at Axosoft. It underwent many design changes, big and small, in the pursuit of improving user acquisition. To support those designs, a robust CSS/HTML framework was designed and built by me, called Axostyle. The result is a flexible, modern, and hand-coded product site.

GitKraken.com

Marketing Web Design CSS Architecture Branding HTML5 Video

As GitKraken neared its v1.0 release, it needed a website and branding that came together to reflect the out-of-the-box aspect of the Git client. GitKraken.com drew on several cutting-edge techniques and patterns to create a unique style, including CSS and video. It was also an opportunity to extend the Axostyle framework to a different product and brand, so that GitKraken could stand out on its own, yet still acknowledge itself as a product of Axosoft.
