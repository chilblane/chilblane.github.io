Selected work

Axosoft v17

Mockups on Behance Axosoft v17 video

I created a series of mockups exploring what the Axosoft product could look like with updated branding and design language, and some UX and UI improvements. The product team decided to incorporate many of these changes into the version 17 release of Axosoft, as well as using parts of the CSS architecture I wrote for Axosoft's websites.

Axosoft Blog

View blog

The Axosoft blog redesign was a fun and challenging project. My goal was to modernize the home of Axosoft’s new content strategy, while adhering to the principles that established their online brand. I worked with the in-house Wordpress developer to come up with a theme using Axosoft's CSS architecture that was not only a great experience for readers, but easy to use for editors, too.

Axosoft support sites

Axosoft Support GitKraken Support

Axosoft had a pressing need for product documentation that was easy to navigate and search, and easy to maintain solely by their support team. We identified a static site generator and server that compiled Markdown articles into pages and adapted it to the CSS architecture. A sister site for GitKraken was later released.
