Axosoft Blog

Wordpress Blog UX CSS Architecture Content Strategy

The Axosoft blog redesign was a fun and challenging project. My goal was to modernize the home of Axosoft’s new content strategy, while adhering to the principles that established their online brand. I worked with the in-house Wordpress developer to come up with a theme using Axosoft's CSS architecture that was not only a great experience for readers, but easy to use for editors, too.
